## 漏洞报告流程

1. 访问 [TCL 安全响应中心](https://src.tcl.com) 获取最新 CNA 提交流程说明。  
2. 确认产品属于表格内范围。  
3. 提交漏洞信息，包括：
   - 漏洞描述
   - 受影响产品型号与版本
   - 复现步骤或 PoC（如有）
4. 等待 TCL 安全团队确认与分配 CVE ID。

---

**注意：** 产品范围可能会随着 TCL 官方公告更新。请遵守负责任的漏洞披露准则。
